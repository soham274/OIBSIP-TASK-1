# Iris Flower Classification Using Machine Learning

## Overview

This project uses Machine Learning techniques to classify iris flowers into different species based on their sepal and petal measurements.

## Objective

To develop a machine learning model that accurately predicts the species of an iris flower using its physical characteristics.

## Tools and Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## Algorithm Used

### K-Nearest Neighbors (KNN)

KNN classifies flowers based on the majority class of their nearest neighboring samples.

## Steps Performed

1. Loaded the Iris dataset.
2. Explored and analyzed the dataset.
3. Selected input features and target variable.
4. Split the dataset into training and testing sets.
5. Applied feature scaling.
6. Trained the KNN model.
7. Predicted flower species.
8. Evaluated model performance.
9. Visualized results using a confusion matrix.

## Outcome

The model successfully classified iris flowers with high accuracy and demonstrated the effectiveness of supervised machine learning classification techniques.
